Gita AI Guide – AI-Based Youth Guidance Platform
• Focused on building a practical AI-guided system to bridge spirituality and modern decision-making for youth.
• Designed and developed a full-stack web application that provides life and career guidance inspired by Bhagavad Gita principles.
• Built responsive frontend using React.js and backend using Flask with RESTful API integration.
• Implemented a rule-based NLP system using keyword scoring and fuzzy matching to simulate intelligent chatbot responses.
• Developed 5 core modules: Mental Wellness, Career Guidance, Ethical Decision Making, Conflict Resolution, and Leadership.
• Created a structured JSON-based knowledge system for dynamic and scalable response generation.
• Enhanced response accuracy by handling real-world inputs, typos, and variations in user queries.
• Successfully deployed the application on cloud platforms (Render & Vercel) with GitHub CI/CD integration.
